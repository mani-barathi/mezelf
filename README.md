  ```sh

      SENDER_EMAIL=YOUR_EMAILID_WHICH_IS_REGISTERED_WITH_SENDGRID_WHILE_CREATING_API_KEY
      RECEIVER_EMAIL=EMAILID_IN_WHICH_YOU_WANT_RECEIVE_EMAIL
  ```
- Start the development server
  ```sh
       npm start
  ```
